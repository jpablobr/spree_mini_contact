SpreeMiniContact

Basically the easiest way to implement a contact form.

Installation

Put the following line into your gemfile:

gem 'spree_mini_contact'

Install it.

bundle install

Add the migrations.

rake spree_mini_contact:install

Run the migration.

rake db:migrate

Done SRLY!

It will save the messages in the database so they will also be available in the admin interface (/admin/contacts).

Set properly the mail method in the admin area (/admin/mail_methods)
